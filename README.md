# Шаблон масштабируемой HTML-страницы

Небольшой шаблон с разделением на CSS и JS для быстрого старта.

Как использовать:

1. Откройте `index.html` в браузере (двойной клик или через локальный сервер).
2. Файлы:
   - `index.html` — основная страница
   - `css/styles.css` — стили
   - `js/app.js` — простая логика (меню, модал, валидация формы)

Ничего дополнительно устанавливать не нужно.
